# PivotTable
This repository provides a python implementation to visualize patterns learned by machine learning models.

## Dependencies
The Code was tested with following packages:
   * Numpy Version 1.26.3
   * itertools Version 3.11.7
   * math Version 3.11.7
## Usage
An example notebook is placed in the notebooks folder.
